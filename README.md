# Golf Cam Replay

Monorepo for the Golf Cam Replay project.

## Structure

- **`web/`** — Marketing website (Next.js)
- **`app/`** — Desktop swing capture application (Python/PyQt6)

## Getting Started

### Web

```bash
cd web
npm install
npm run dev
```

### App

```bash
cd app
pip install -r requirements.txt
python swing_capture.py
```
