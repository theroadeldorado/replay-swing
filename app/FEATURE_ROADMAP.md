# Feature Roadmap - Golf Swing Replay

Prioritized feature list for building the best affordable PC swing replay app.

## Priority 1 - Core Reliability

- [ ] Multi-camera recording & playback (fix current broken state)
- [ ] iPhone camera support (DroidCam iOS + MJPEG direct)
- [ ] Reliable audio trigger with adaptive threshold
- [ ] Session management (browse/load previous sessions)

## Priority 2 - Analysis Features

- [ ] Swing tempo measurement (backswing/downswing timing)
- [ ] Frame-by-frame annotation (text labels on frames)
- [ ] Swing overlay comparison (ghost overlay of two swings)
- [ ] Auto-trim clips to swing motion only

## Priority 3 - Camera & Video

- [ ] Phone camera auto-discovery (mDNS/Bonjour)
- [ ] Camera resolution/quality presets
- [ ] Video export (MP4 with overlays baked in)
- [ ] Slow-motion recording (60fps+ support)

## Priority 4 - UI/UX Polish

- [ ] Onboarding wizard (first-run camera setup)
- [ ] Camera preview in settings dialog
- [ ] Drag-to-reorder cameras
- [ ] Fullscreen playback mode
- [ ] Better multi-camera grid with resizable panes

## Priority 5 - Advanced

- [ ] Cloud backup (Google Drive / OneDrive)
- [ ] Share clips (generate shareable links)
- [ ] Pro swing reference library
- [ ] AI swing analysis (pose estimation)
